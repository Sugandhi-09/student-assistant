# 🎓 Student Toolkit

Welcome to **Student Toolkit**, a menu-driven Python application designed to help students with common academic calculations.

## ✨ Features

- 🧮 Basic Calculator
- 📊 Percentage Calculator
- 📚 Attendance Calculator
- 🎓 CGPA Calculator

## 🛠️ Built With

- Python

## 🚀 How to Run

1. Download or clone this repository.
2. Open the `main.py` file in any Python IDE (such as IDLE, VS Code, or PyCharm).
3. Run the program.
4. Select the desired option from the menu and follow the prompts.
